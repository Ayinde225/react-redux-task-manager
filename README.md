# React + Redux Task Manager

A task manager built with React and Redux to practice global state management with actions and reducers.

## Features

- Add new tasks to a central Redux store
- Toggle a task between done / not done
- Edit existing task text
- Filter tasks by status
- Clean separation of `actions`, `reducers` and presentational components (`Task`, `TaskList`, `AddTask`)

## Getting started

```bash
git clone https://github.com/Ayinde225/react-redux-task-manager.git
cd react-redux-task-manager
npm install
npm start
```

Then open http://localhost:3000 in your browser.
